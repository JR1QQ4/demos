JavaScript

什么是 JavaScript 语言？

JavaScript 是一种具有函数优先的轻量级，解释型或即时编译型的编程语言。

JavaScript 是一种基于原型编程、多范式的动态脚本语言，并且支持面向对象、命令式和声明式（如函数式编程）风格。

JavaScript 既是一个 面向过程的语言 又是一个 面向对象的语言。

JavaScript 是一种嵌入式（embedded）语言。JavaScript 本身不提供任何与 I/O（输入/输出）相关的 API，都要靠宿主环境（host）提供。
